# Генератор карточек клиентов

Простая утилита на Python для создания HTML-карточек с данными клиента и настройкой стилей. Подходит для визуализации заказов на мясо: какой клиент, какую часть хочет, в каком объёме и с какими пожеланиями.

## Возможности
- Загрузка данных из JSON или CSV.
- Настройка цветов, шрифтов, радиуса скругления и размера карточек через отдельный JSON.
- Экспорт в один HTML-файл, готовый для печати или отправки.

## Быстрый старт
1. Установите Python 3.10+.
2. Запустите генератор на примере:
   ```bash
   python main.py customer_cards/data/clients.sample.json \
     --style customer_cards/data/style.sample.json \
     --output cards.html
   ```
3. Откройте `cards.html` в браузере.

## Формат входных файлов
### Клиенты (JSON или CSV)
- **Обязательные поля:** `name`, `meat`, `cut`, `weight`.
- **Необязательное поле:** `notes`.
- Можно использовать русские ключи (`имя`, `мясо`, `часть`, `количество`, `комментарий`).

Пример JSON: см. [`customer_cards/data/clients.sample.json`](customer_cards/data/clients.sample.json).

### Стили (JSON)
Поддерживаемые параметры указаны в [`customer_cards/data/style.sample.json`](customer_cards/data/style.sample.json). Любой параметр можно опустить — будет подставлено значение по умолчанию.
