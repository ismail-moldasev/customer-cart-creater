# Одностраничный генератор карточек

Веб-приложение, которое моментально создаёт красивые карточки для клиентов или продавцов. Заполните форму, нажмите «Создать карточку» — запись появится на странице без перезагрузки.

## Запуск
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите сервер:
   ```bash
   python main.py
   ```
3. Откройте в браузере [http://localhost:8000](http://localhost:8000) и создавайте карточки прямо в окне.

## Возможности
- Выбор роли: **Клиент** или **Продавец**.
- Поля для имени, возраста, пола, телефона, суммы оплаты или объёма поставки, а также комментария.
- Кнопка «Создать карточку» добавляет карточку в общий список, «Очистить список» удаляет все карточки.
- Современный тёмный интерфейс с акцентами и адаптивной сеткой карточек.

## Структура проекта
- `main.py` — минимальный сервер Flask, который отдаёт одностраничное приложение.
- `templates/index.html` — разметка страницы с формой и блоком карточек.
- `static/style.css` — оформление страницы и карточек.
- `static/app.js` — логика добавления карточек без перезагрузки.
